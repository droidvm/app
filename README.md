#### 软件介绍
安卓手机免root运行proot-linux桌面系统，把手机当电脑，<br>
支持安卓8.0以上的任意设备，包括手机、平板、电视、盒子、车机，<br>
用于3D打印控制、制作启动U盘、便携办公、编程学习、游戏娱乐、少儿电脑、户外应急电脑。

#### 安装教程
1.  下载最新版apk安装包, 然后安装并启动
2.  按照屏幕弹窗提示安装虚拟系统

#### 使用说明
1.  开始菜单 已集成多种常用功能，如启动sshd、ftpd文件共享、webtty
2.  软件管家中 已收录常用软件的自动化安装脚本
3.  桌面上有教程目录


#### 有何限制(不影响日常使用)
1. 网络管理部分<br>
不能使用 1024 以下的端口号<br>
不能管理防火墙

2. 外设访问部分<br>
外设目前仅支持 usb 串口, 用于连接3D打印机<br>
不支持外接 usb 办公打印机<br>
不支持虚拟系统中使用设备的摄像头、麦克风<br>

3. 软件适配相关<br>
不支持 docker, 但可以安装 udocker<br>
不支持 systemd, 但 service 指令可用<br>
不支持 appimage、flatpak 等安装包格式<br>
exe兼容性非常非常有限，基本等于不支持

#### 同类应用比较
1. 无需root、不用刷机，不破坏手机的权限系统，手机仍可以放心的用于日常支付和拨打电话
2. 不用敲指令，省心不折腾
3. 直绘屏幕画面，不需要vnc中转，显示效率更高
4. 虚拟系统存储空间占用小，整个虚拟系统安装完图形界面仅占用1.5GB左右存储空间
5. 虚拟系统的软件下载仓库自动切换成大陆仓库，下载速度快
6. 支持满屏无线投屏
