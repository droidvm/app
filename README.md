#### 安装教程
1.  下载最新版apk安装包, 然后安装并启动
2.  按照屏幕弹窗提示安装虚拟系统

#### 使用说明
1.  开始菜单 已集成 sshd、webtty、ftpd文件共享
2.  软件管家中 已收录常用软件的自动化安装脚本
3.  桌面上有教程目录


| |平行对比|
|--|:--|
|1|超快启动, k40上1秒见桌面|
|2|无需越狱、不用刷机，不破坏手机权限系统<br>手机仍可以放心的用于日常支付和拨打电话|
|3|省心不折腾<br>不用敲指令，按照屏幕弹窗提示点确定即可|

|5|支持满屏无线投屏<br>外接键鼠|

|7|虚拟系统为 linux, 依赖于 proot, 安装虚拟系统时将会自动下载此工具|
|8|apk只有约4MB, 装完虚拟系统约占用1.5GB|


#### 有何限制(不影响日常使用)
1. 网络管理部分<br>
不能使用 1024 以下的端口号<br>
不能管理防火墙

2. 外设访问部分<br>
外设目前仅支持 usb 串口, 用于连接3D打印机<br>
不支持外接 usb 办公打印机<br>
不支持虚拟系统中使用设备的摄像头、麦克风<br>

3. 软件适配相关<br>
不支持 docker, 但可以安装 udocker<br>
不支持 systemd, 但 service 指令可用<br>
不支持 appimage、flatpak 等安装包格式<br>
exe兼容性非常非常有限，基本等于不支持
