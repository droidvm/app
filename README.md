#### 安装教程
1.  下载最新版apk安装包, 然后安装并启动
2.  按照屏幕弹窗提示安装虚拟系统

#### 使用说明
1.  开始菜单 集成 sshd、webtty、ftpd文件共享
2.  软件管家 已收录常用软件的自动化安装脚本
3.  桌面上有教程目录
4.  [多进程应用-防杀保活方法](https://gitee.com/droidvm/app/blob/master/z_unfreeze.md)

#### 有何限制(不影响日常使用)
1. 架构部分<br>
权限低，整体受限于安卓对app的权限限制<br>
默认使用安卓内置的命令行系统(toybox, 指令少)<br>
安装虚拟系统时依赖 proot(会自动下载此工具)<br>

2. 网络管理部分<br>
不能使用 1024 以下的端口号<br>
不能管理防火墙

3. 外设访问部分<br>
外设目前仅支持 usb 串口, 用于连接3D打印机<br>
不支持外接 usb 办公打印机<br>
不支持虚拟系统中使用设备的摄像头、麦克风<br>

4. 软件适配相关<br>
不支持 docker, 但可以安装 udocker<br>
不支持 systemd, 但 service 指令可用<br>
不支持 appimage、flatpak 等安装包格式<br>
exe兼容性非常非常有限，基本等于不支持
